# TravelTours – Travel Website

TravelTours is a simple, responsive travel website built using HTML5, CSS3, Bootstrap 5, and Font Awesome. It includes a home page, about page, and contact page suitable for a basic travel agency or portfolio project.

## Features

- Full-screen hero section with image background and Bootstrap carousel.
- Sticky dark navbar with brand logo and links to Home, About, and Contact.
- Featured Tours section using Bootstrap cards (e.g., Maldives Paradise tour).
- Stats section showing travelers, countries, tours, and rating.
- Testimonials carousel with customer reviews and avatars.
- About page with company story, stats, services, and hero banner.
- Contact page with contact info, business hours, form, and success modal.
- Shared footer with quick links and social media icons.
- Fully responsive layout using the Bootstrap grid system.

## Files

- `index.html` – Home page with hero carousel, stats, featured tours, testimonials, and footer.
- `about.html` – About page with hero banner, story, stats, services, and footer.
- `contact.html` – Contact page with hero banner, contact info, form, success modal, and footer.

## Technologies Used

- HTML5 and CSS3  
- Bootstrap 5.3.3 (CDN)  
- Font Awesome 6.4.0 (CDN)  
- Unsplash image URLs for backgrounds and photos
